## Desktop app (Tauri)

Download installers from the [Releases page](https://github.com/Ovilli/RavenswatchModManager/releases).

- Windows: `RSMM-x64.msi`
- macOS: `RSMM-universal.dmg`
- Linux: `RSMM.AppImage` or `rsmm.deb`

## Browser

Use [rsmm.dev](https://rsmm.dev) to browse the registry and manage your account. The
browser app cannot install mods directly — install the desktop app or use the CLI for that.

## CLI (advanced)

```sh
git clone https://github.com/Ovilli/RavenswatchModManager
cd RavenswatchModManager
./rsmm doctor
```

Requires Python 3.11+.
