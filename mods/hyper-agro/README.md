# Hyper Aggro

This mod makes enemies much more eager to detect, engage, and attack by widening the default enemy height thresholds and nearby-enemy pressure values, then boosts enemy damage and health directly.

It patches these fields:

- Enemy_Detection_Min_Height_Default
- Enemy_Detection_Max_Height_Default
- Enemy_Fighting_Min_Height_Default
- Enemy_Fighting_Max_Height_Default
- Enemy_Attacks_Min_Height_Default
- Enemy_Attacks_Max_Height_Default
- EnemyAround_LongRange_Count
- EnemyAround_ShortRange_Count
- NGP_Enemy_Damage_Modifier
- NGP_Enemy_Max_Health_Modifier

Install or refresh it with `./rsmm apply`.
